Elementary Cellular Automata are a very simple kind of simulation using binary cells. Each row is a generation, advancing through time from top to bottom.

Each new cell's state (0 or 1) is determined by the previous cell directly above, above-left, and above-right of it. A list of rules determines the new state.

For more information, see Wolfram MathWorld and Wikipedia

Some interesting rules of note are Rule 110 (which is Turing Complete) and Rule 30 (which is used to generate randomness). I personally like how Rule 57 looks with the "only middle" option.
